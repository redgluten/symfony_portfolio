Symfony
=======

A Symfony student project.

## Dependancies
- Framework [Symfony 2.6](http://symfony.com/doc/current)
- [Nodejs 0.12.1](https://nodejs.org/) is used for Gulp & Bower
- [Gulp 3.8](http://gulpjs.com/) as a build system for Sass, minification, etc
- [Bower](http://bower.io) for front-end assets libraries
- [Sass](http://sass-lang.com) as a CSS-preprocessor
- [Autoprefixer](https://github.com/postcss/autoprefixer) as a gulp plugin
